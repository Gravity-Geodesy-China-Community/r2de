# DataEngineerInterview
各位读者朋友们，本教程为非计算机专业快速入门数据工程师行业的教程。

本教程主要内容包括以下七个部分：

第 1 章主要讲述计算机网络，主要是TCP/IP 协议，包含HTTP、套接字等相关知识
第 2 章主要介绍操作系统，主要是Linux系统相关知识
第 3 章主要讲述数据结构与算法
第 4 章主要介绍Java语言基础以及常用框架
第 5 章主要介绍大数据常用框架与开发
第 6 章主要介绍深度学习的基础以及应用实例
第 7 章手把手带你实战项目
